Day 4

Mixed Mode Arithmetic

  • Integers always yield integers, may truncate decimals
  • Doublies yield doubles, overrule integers
  • Automatically promotes to large data type or a floating point
  1. int a = 3/2 * 5; //five
  2. double b = 3/2 *5; //5.0
  3. double c = 3.0/2 * 5; //7.5
  4. int d = 3.0/2 * 5; //7
  5. double e = 3.0/2 * 5 //ERROR
  6. double f = 3/2* 5.0; //5.0

Remainder Operator % (modulus)

  1. 3 % 2 = 1
  2. 16 % 12 = 4
  3. 35 % 5 = 0
  4. 17 % 2 = 1
  5. 63 % 24 = 15

Casting

  • Converts from one data type to another
  • int x = (int) 3.14; //x = 3
  • int y = (int) (3/2 * 5.0); //5
  • int z = (int)3/2 *5;//5.0, ERROR
